**کارهایی که من انجام دادم:**
- یک فایل `.gitignore` برای پروژه ساختم تا فایل‌های سیستمی و غیرضروری وارد مخزن نشوند.

- تمامی فایل‌های پروژه را به مرحله Stage بردم (`git add .`).
- اولین کامیت (Commit) را با پیام مناسب ثبت کردم.

**کارهایی که شما باید انجام دهید (اتصال به گیت‌هاب):**

1. **ایجاد مخزن در گیت‌هاب:**
   - وارد اکانت GitHub خود شوید.
   - روی دکمه **New** (یا علامت + در بالا سمت راست) کلیک کنید.
   - نام مخزن را مثلاً `Soheili-MDF` بگذارید و روی **Create repository** کلیک کنید.

2. **اتصال مخزن محلی به گیت‌هاب:**
   - آدرس مخزنی که ساختید را کپی کنید (شبیه این: `https://github.com/YourUsername/Soheili-MDF.git`).
   - در ترمینال (در همین مسیر پروژه)، دستورات زیر را به ترتیب وارد کنید:

   ```bash
   # اضافه کردن آدرس مخزن گیت‌هاب به پروژه محلی
   git remote add origin https://github.com/YourUsername/Soheili-MDF.git

   # تغییر نام شاخه اصلی به main (استاندارد جدید گیت‌هاب)
   git branch -M main

   # آپلود (Push) کدها به گیت‌هاب
   git push -u origin main
   ```

3. **بعد از هر تغییر در آینده:**

**۱. یکپارچه‌سازی داده‌های صفحه اصلی (Dynamic Home Data)**
- **چی کار کردم:** قبلاً محصولات و پروژه‌های صفحه اصلی به صورت دستی (Hardcoded) نوشته شده بودند. الان آن‌ها را حذف کردم و با استفاده از JavaScript، اطلاعات را مستقیماً از لیست محصولات و پروژه‌های موجود در `main.js` می‌خوانم.
- **چرا این کار مهم بود:** با این کار، اگر محصولی را در فایل جاوااسکریپت تغییر دهیم یا اضافه کنیم، صفحه اصلی هم به صورت خودکار به‌روز می‌شود و نیازی به تغییر دستی HTML نیست (رعایت اصل Don't Repeat Yourself).

**۲. قابلیت ارسال مستقیم به واتساپ (WhatsApp Integration)**
- **چی کار کردم:** به فرم مشاوره در صفحه تماس، دکمه «ارسال به واتساپ» اضافه شد.
- **چطور کار می‌کند:** وقتی کاربر فرم را پر می‌کند یا دکمه ثبت را می‌زند، این دکمه ظاهر می‌شود. با کلیک روی آن، تمام اطلاعات فرم (نام، شماره، موضوع و متن درخواست) به صورت خودکار قالب‌بندی شده و در واتساپِ مدیر سایت (با شماره نمونه) باز می‌شود.
- **چرا این کار مهم بود:** این قابلیت مسیر تبدیل بازدیدکننده به مشتری را بسیار سریع‌تر و راحت‌تر می‌کند.
- **فایل‌های تغییر یافته:** [contact.html](file:///c:/xampp/htdocs/Soheili-MDF/contact.html) و [main.js](file:///c:/xampp/htdocs/Soheili-MDF/assets/js/main.js).

- لیست محصولات با فیلتر و جستجو
- جزئیات محصول با پیشنهادات مشابه
- لیست پروژه‌ها و خدمات طراحی
- فرم تماس هوشمند با قابلیت کپی و ارسال به واتساپ
- بهینه‌سازی شده برای موبایل و تبلت
- سئو شده و دارای انیمیشن‌های مدرن
